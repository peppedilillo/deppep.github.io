Sculare tuo nonno a Las Vegas: una simulazione¶

Premetto che non ho idea di quanto questo gioco sia diffuso. Spero non sia troppo locale. Di sicuro nella mia città del sud Italia, a Natale, va forte. Il gioco si chiama Las Vegas ed è una variante di merda del poker tradizionale. Le regole le trovate qui.

Brevemente. Si gioca con un mazzo di carte francesi da 52 carte. Si distribuiscono 5 carte per giocatore e 5 carte per il banco. Per ricevere le carte si paga un puntata obbligatoria, che serve a riempire il piattino iniziale. I giocatori guardano le carte che hanno ricevuto, e vengono scoperte un tot di carte del banco. I giocatori scartano le carte con lo stesso valore numerico delle carte scoperte dal banco. A questo punto parte un giro di puntate, check, fold e tutta la solita tiritera, come in una partita di poker. Rinse and repeat, fin quando tutte le carte del banco sono scoperte. Se un giocatore rimane senza carte vince tutto il piattino--in questo caso si dice che ha fatto "Las Vegas". Altrimenti, i giocatori mostrano le carte dalla loro mano, e vincono i giocatori che hanno realizzato il punteggio più basso ed il punteggio più alto (50/50 se i vincitori sono due). Tutte le figure (Jack, Queen, King) valgono 10 punti. Sul wiki riportano una fantomatica regola per cui tutti gli assi oltre il primo valgono 11--personalmente non l'ho mai sentita, quindi facciamo come non esista.

In questo notebook simuleremo tanti giochi semplificati di Las Vegas e proveremo a tirare fuori dei numeri utili a valutare l'andamento di una partita, nella speranza di riuscire a fare il culo almeno a tuo nonno ubriaco.

def compute_scores(hands):
    return list(map(
        lambda hand: np.sum(np.clip(hand, a_min = None, a_max = 10)), 
        hands
    ))

def winning_scores(player_scores):
    if 0 in player_scores: return [0] * player_scores.count(0)
    return sorted(player_scores)[::num_players - 1]

def play_a_game(num_players, rng):
    deck = np.array([i for i in range(1, 14) for j in range(4)])
    rng.shuffle(deck)
    starting_hands, rest_of_deck = deck[:5 * num_players], deck[5 * num_players:]
    starting_hands = starting_hands.reshape(num_players, 5)
    u, ind = np.unique(rest_of_deck, return_index=True)
    dealer = u[np.argsort(ind)][:5]
    player_hands = list(map(
        lambda hand: hand[~np.isin(hand, dealer)],
        starting_hands,
    ))
    return compute_scores(player_hands)

Mano iniziale¶

Il momento più delicato del gioco è sicuramente quello della prima puntata. Abbiamo ricevuto una mano e ci troviamo a valutare quanto buona questa sia. Se la mano è buona, è molto probabile che vorremo continuare a giocare. Se la mano è meh, potremmo volerci ritirare, limitando le perdite alla puntata di entrata.

def winners(player_scores):
    if 0 in player_scores: 
        ws = [0] * player_scores.count(0)
    ws = sorted(player_scores)[::num_players - 1]
    wargs = np.argwhere(np.isin(player_scores, ws)).T[0]
    return ws, wargs

def multiplicity(hands):
    return list(map(lambda h: len(h) - len(set(h)), hands))

def play_a_game(num_players, rng):
    deck = np.array([i for i in range(1, 14) for j in range(4)])
    rng.shuffle(deck)
    starting_hands, rest_of_deck = deck[:5 * num_players], deck[5 * num_players:]
    starting_hands = starting_hands.reshape(num_players, 5)
    u, ind = np.unique(rest_of_deck, return_index=True)
    dealer = u[np.argsort(ind)][:5]
    player_hands = list(map(
        lambda hand: hand[~np.isin(hand, dealer)],
        starting_hands,
    ))
    return player_hands, starting_hands
